## Overview
The CanvasAI shell is designed as a high-productivity workspace that prioritizes maximum screen real estate for the design canvas. It utilizes a persistent, collapsible left-hand sidebar for primary navigation and a contextual top bar for project-specific metadata and workspace controls.

## Navigation Items
- **Dashboard**: A centralized hub for accessing recent projects, design activity, and performance metrics.
- **AI Workspace**: The primary creative interface featuring the natural language prompt engine and real-time generation canvas.
- **Design Systems**: A management suite for defining brand identities, including color palettes, typography, and component libraries.
- **Asset Library**: A searchable repository of generated SVG, Figma-compatible files, and brand imagery.
- **Shared Projects**: A collaborative space for team-based iterations, version history, and stakeholder feedback.
- **Integrations**: Configuration panel for connecting to third-party tools like Figma, Slack, or GitHub for export automation.
